Product information management means managing the information required to market and sell products through distribution channels. A central set of product data can be used to share/receive information with media such as web sites, print catalogs, ERP systems, PLM systems and electronic data feeds to trading partners.

Before we are going to visit this web portal, there are certain features need to be dislosed to satisfy the user interaction and experince in a more trendy way.

  • Dashboard is equiped with all CURD operation controls like ADD, EDIT and DELETE.
  • All the operations are authenticated with secured login.
  • Dashboard contents are read only if user is not authenticated.
  • Header is having navigational routes with short login information with logout option.
  • Carousel is meant to rotate top 3 products in the list, user can use corresponding buttons to go for next and prev options.
  • Top Rated button on carousel is meant to see the graphical view of the top products with their view ratings on Popup container.
  • Graph view is having a custom navigation link to update the ratings according to the selection.
  • Customized Rating page is displaying dynamic controls with default ratings, once user change the selected values and click on Update rates, it will update the main product data.
  • Login and Registration feature is available on the portal.
  • Global product search option is available on top of product list
